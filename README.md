# ReactEasyCounter

ReactEasyCounter is a simple counter application built using React and `create-react-app`. This application includes a counter that you can increment or decrement by clicking on the "+" and "-" buttons. You can also reset the counter to zero by clicking on the "Reset" button.

## Installation and Setup

To install and setup this app, you need to have Node.js and npm installed on your machine. Then, you can clone this repository and install the dependencies using npm:

```bash
git clone https://github.com/AmirrMahmoudi/ReactEasyCounter.git
cd ReactEasyCounter
npm install
```

After the installation is complete, you can start the application:

```bash
npm start
```

The application will start and you can access it at `http://localhost:3000` in your web browser.

## Usage

The application includes a counter displayed on the screen. You can increment the counter by clicking on the "+" button, decrement it by clicking on the "-" button, or reset it to zero by clicking on the "Reset" button.
